# Foodpanda Clone

This is a frontend clone of the Foodpanda website created to practice layout building and responsive web design. The project focuses on replicating the visual structure and responsiveness of the original website using modern CSS techniques.

## live link

[Live Website](https://foodpandas.vercel.app/)


## Features

- Responsive layout for mobile, tablet, and desktop
- Section-based page structure
- Restaurant and food listing layout
- Clean and structured UI
- Uses modern CSS layout techniques

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries


## How to Run

1. Download or clone the project
2. Open the project folder
3. Open `index.html` in any modern browser

## Purpose

This project was built to improve frontend development skills, especially responsive design and real-world website layout cloning.
